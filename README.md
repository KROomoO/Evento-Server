## 💡 서비스 소개
이벤트 조회 웹 애플리케이션
>Evento는 서울시에서 제공하는 공공데이터를 기반으로 개발되었습니다.<br/>
>서울시에서 열리는 행사, 공연, 축제 정보를 쉽게 조회할 수 있는 플랫폼을 제공합니다.
- 자치구별, 월별, 유/무료 이벤트 조회
- 이벤트 상세정보
- 마이페이지: 스크랩 이벤트 조회
<br/>
